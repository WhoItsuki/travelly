Modern and easy-to-use travel booking platform for exploring amazing destinations and managing your travel experiences.

## 🌍 Overview

Travelly is a frontend web application that makes booking travel packages simple and enjoyable. Users can browse various destinations, create accounts, make bookings, and get in touch with customer support.

## ✨ Features

- 🏠 **Home Page**: Browse featured destinations and quick booking access
- 📝 **User Authentication**: Sign up and login functionality
- 🎫 **Booking System**: Easy-to-use booking management
- 📧 **Contact Support**: Get in touch with the customer service team
- 🎨 **Responsive Design**: Beautiful UI/UX experience across devices
- 🏝️ **Destination Showcase**: Featured travel packages including:
  - Penang Island
  - Perhentian Island
  - Kuala Terengganu

## 🚀 Getting Started

1. **Clone or download the project** to your local machine
2. **Open** `html/index.html` in your web browser
3. **Explore** the website and try booking a destination!
